Ready to sing? RPCS3 lets you set up to 3 microphones to sing with.  
Just about any microphone can be used to sing.

To get started setting up microphones, make sure Rock Band 3 isn't running.  
**<a href="../quickconfig/" target="_blank" rel="noopener noreferrer">[Also, make sure you have installed a Quick Configuration Profile from the previous page]</a>**.  

<Steps>
1. **Right click on Rock Band 3** in RPCS3, then click **`Change Custom Configuration`**.
     > <Image src={rpcs3customconfigchange} alt="RPCS3's right click menu." />

2. Switch to the `Audio` tab.
     > <Image src={audiom} alt="Audio settings" />

3. ![A tan square with a solid outline.](/assets/rb3pc/config/smalltan.png "Tan Square") **To set microphones up:**
   - **Set `Microphone Type` to `Standard`.**
   - **Select an input device in `Mic1`, `Mic2`, and `Mic3`.**

4. When you're done, click `Apply` then `Save custom configuration`.
</Steps>

<details>
<summary>Adjusting microphone volume</summary>
If the game is having issues detecting your voice, try adjusting the mic levels in Windows.  
Rock Band 3 requires a microphone volume that isn't too loud nor too quiet.  
<Steps>
1. Right click on speaker icon in the bottom right of Windows' taskbar and click on `Sound settings`.
     > <Image src={micstartmenu} alt="A right click menu from the speaker icon in the taskbar. The cursor hovers over 'Sound settings'." />
2. Scroll down to the `Input` section and click on the microphone having issues with levels.
     > <Image src={micwindows} alt="Windows' Sound settings. A cursor is hovering over an input device named 'Mic'." />
3. Adjust the `Input volume` slider until detection improves.
     > <Image src={micvol} alt="Windows' Sound settings within an input device's properties. A cursor is adjusting the input volume for an input device named 'Mic'." />
</Steps>
</details>
<Aside type="danger">
**Make sure you have [something set up to control your game](../../controllers/main#gamepads).** You can't select a song to sing otherwise.
</Aside>
