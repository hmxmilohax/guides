### Setup
<Tabs>
  <TabItem label="Windows">
<Steps>
1. Connect your RB3 MIDI PRO Adapter and switch it to Drum mode. Simply set the middle switch to left-most `D` position.  
Please note that you need to do this even if you already did it for the other two modes.  
    > <Image src={mpadrm} alt="A MIDI PRO Adapter with the middle switch set to 'D'." />

2. Now, **<a href="https://zadig.akeo.ie/" target="_blank" rel="noopener noreferrer">[go to Zadig's website]</a>** and **download the latest version, then open it** up.

3. Click on **`Options`** then **`List All Devices`**.  
    > <Image src={zadiglistall} alt="Zadig showing 'List All Devices' under 'Options' highlighted." />

4. You should now see devices listed. **Switch it to your Rock Band 3 Pro Instrument**.  
In this example, we're using the Mustang Pro Guitar, which shows up as `Harmonix RB3 Mustang Guitar for PlayStation® 3`.  
    > <Image src={zadigsel} alt="A screenshot of Zadig showing 'Harmonix RB3 Mustang Guitar for PlayStation® 3' highlighted in the devices listed." />

5. **After selecting the right device, you should see the option to replace the driver.  
_MAKE SURE YOU ARE REPLACING THE DRIVER ONLY FOR THE PRO GUITAR/KEYBOARD/MIDI PRO ADAPTER!!_**  
Click `Replace Driver`.
    > <Image src={zadigreplace} alt="Zadig with 'Replace Driver' highlighted." />

6. A warning will appear.  
**Again, make sure you have selected your RB3 Pro instrument's receiver or adapter.**  
After you have made sure, **click** **`Yes`**.  
    > <Image src={zadigwarning} alt="A screenshot of Zadig warning the user that they're about to modify a system driver, with 'Yes' highlighted" />

7. It will then install the driver. As the program says, it may take a few minutes.  
    > <Image src={zadigprogress} alt="A screenshot of Zadig in the middle of a driver install." />

8. If everything goes well, you will get this message:  
    > <Image src={zadigdone} alt="Zadig telling the user that the driver was installed successfully with 'Close' highlighted." />

9. **Close Zadig**.

10. Open up Rock Band 3 in RPCS3 and connect your MIDI Pro Adapter.  
You should see it automatically assign a player number.  
    Likewise, in Rock Band 3, you will see the instrument ready to join.  
    From this point, you can just connect your RB3 MIDI Pro Adapter whenever you want without having to repeat this setup process!
</Steps>

<details>
<summary>Uninstalling</summary>
Applying a Zadig driver on a device will make it only work on applications with passthrough. This may cause issues with Clone Hero or YARG.  
Thankfully, reverting this is very easy!
<Steps>
1. Open the Start menu and search for `Device Manager`. Open it.
    > <Image src={startdevman} alt="Windows' Start menu, with a result of 'Device Manager'." />

2. Scroll down until you find the `Universal Serial Bus devices` category.  
Expand it and right click the instrument you applied the Zadig driver on and then select `Uninstall device`.  
In this example, it's the Pro Guitar (`Harmonix RB3 Mustang Guitar for PlayStation® 3`).
    > <Image src={devman} alt="Device Manager, with a right click menu that popped up. The cursor is over 'Uninstall Device.'" />

3. In the next window that pops up, make sure to tick `Attempt to remove the driver for this device.`  
After that, click `Uninstall` and that's it!  
    > <Image src={devmanun} alt="The device uninstall prompt. 'Attempt to remove the driver for this device' is ticked and the cursor is over 'Uninstall.'" />
</Steps>
When you disconnect then reconnect the device, it will revert back to the original driver.
</details>

  <TabItem label="Linux">
<Aside>This requires admin/sudo permissions.</Aside>
<Steps>
1. Navigate to `/etc/udev/` in your file browser.  
Enter the `rules.d` folder.  
If it doesn't exist, make it.
    > <Image src={doludev} alt="A file browser in the 'udev' folder, with a cursor highlighting and clicking on a folder called 'rules.d'." />

2. Create a new file.
    > <Image src={dolnewfile} alt="Creating a new empty file within the 'rules.d' folder." />
    Name it `99-ps3-peripherals.rules`.  
    > <Image src={dolps3rules} alt="Naming the new file." />

3. Open the file in a text editor.  
    Add an entry for your MIDI Pro Adapter:  
    ```
    SUBSYSTEM=="usb", ATTRS{idVendor}=="1bad", ATTRS{idProduct}=="3430", MODE="0666"
    ```
    In this example, a PS3 Mustang and PS3 Keyboard are being added.  
    When you're done, save your edits to the file.
    > <Image src={kateps3} alt="A text editor showing the '.rules' file open. There are two entries in the text file." />

4. Run `sudo udevadm control --reload-rules` in a terminal or reboot your system.
    > <Image src={termreloadrules} alt="A terminal with the command typed in." />

5. Open up Rock Band 3 in RPCS3 and connect your MIDI Pro Adapter.  
    You should see it automatically assign a player number.  
    Likewise, in Rock Band 3, you will see your MIDI Pro Adapter ready to join.
</Steps>

  <TabItem label="macOS">
By default, macOS will attempt to load its own drivers when trying to use passthrough instruments on RPCS3.  
Luckily, this is easy to fix!

<Steps>
1. First, [[click here download the GCAdapterDriver 1.8 beta]](https://secretkeys.io/gcadapterdriver/GCAdapterDriver-1.8-beta.app.zip)
    > <Image src={macgcdl} alt="A '.zip' file of GCAdapterDriver being downloaded." />

2. Drag the extracted `GCAdapterDriver-1.8-beta` application to the `Applications` folder.
    > <Image src={macfnddnd} alt="Dragging GCAdapterDriver to the Applications folder." />

3. In the menu bar, click on the new GCAdapterDriver icon.  
    Click on the slider to try to activate it.
    > <Image src={macgcdis} alt="GCAdapterDriver is disabled." />

4. It will initially block the system extension.  
    Click on `Open Security Preferences`.
    > <Image src={macsysex} alt="A popup alerting that the GCAdapterDriver system extension was blocked. The cursor is over 'Open Security Preferences.'" />

5. Unlock the page by click on the lock icon on the bottom left.  
    Click on `Allow` after this.  
    Don't forget to lock the page again when you're done.
    > <Image src={macauth} alt="The 'Security & Privacy' page in System Preferences. The lock has been clicked on to unlock the page and the cursor hovers over 'Allow.'" />

6. Now, activate GCAdapterDriver's slider in the menu bar now.
    > <Image src={macgcen} alt="GCAdapterDriver is enabled." />
    Open up Rock Band 3 in RPCS3 and connect your MIDI Pro Adapter.  
    You should see it automatically assign a player number.  
    Likewise, in Rock Band 3, you will see your MIDI Pro Adapter ready to join.  
</Steps>
