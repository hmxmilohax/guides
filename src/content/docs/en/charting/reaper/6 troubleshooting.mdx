  <TabItem label="Windows">
* I get "`Error opening devices`" when swapping to WASAPI!
  > <Image src={errordevicewin} alt="REAPER showing an error that says 'There was an error opening the audio hardware. WASAPI Error: could not find input device'." />
  * `Could not find input device`: You either don't have a microphone connected or your formats are (i.e. speakers at 48,000 Hz and microphone at 44,100 Hz).  
  If it's the former, change the "`Input channels`" to "`0`" in the `Device` section of Preferences.  
  If it's the latter, make sure your microphone matches the format of your output device. 
* RBN Preview doesn't load!
  > <Image src={errorpluginwin} alt="REAPER showing a broken RBN Preview." />
  * You really shouldn't be using this anymore but if you insist, download the appropriate Microsoft Visual C++ Redistributable Runtimes.  
  <a href="https://kutt.it/vcpp" target="_blank" rel="noopener noreferrer">[Click here to download them]</a>.  
</TabItem>
<TabItem label="Arch">
* I get "`JACK: error creating client`" when swapping to JACK!
  > <Image src={errorjackarch} alt="REAPER showing an error that says 'There was an error opening the audio hardware. WASAPI Error: could not find input device'." />
  #### Solution A:
  * You have `jack2` installed and need to replace it with `pipewire-jack`.
    * Install `pipewire-jack` by running this command in a terminal.
        ```bash
        sudo pacman -S pipewire-jack
        ```
        > <Image src={archpwjack} alt="A terminal downloading pipewire-jack via Pacman." />
  #### Solution B:
  * Change to a different audio system, like `PulseAudio` or `ALSA`.
    > <Image src={settingsaudioarch} alt="REAPER's Preferences showing the Device section." />
</TabItem>
<TabItem label="Fedora">
* I get a `Script execution` error when opening CAT or VenueGen that says "ModuleNotFoundError: No module named 'tkinter'".
  > <Image src={errorreascriptfed} alt="REAPER showing a broken ReaScript." />
  * Install "python3-tkinter"
    ```bash
    sudo dnf install python3-tkinter
      ```
      > <Image src={fedtermtk} alt="A terminal downloading python3-tkinter via DNF." />
* I get "`Error opening devices`" when swapping to JACK!
  > <Image src={errordevicegnu} alt="REAPER showing an error that says 'Three was an error opening the audio audio hardware'." />
  * Change to a different audio system, like `PulseAudio` or `ALSA`.
    > <Image src={settingsaudioarch} alt="REAPER's Preferences showing the Device section." />
</TabItem>
