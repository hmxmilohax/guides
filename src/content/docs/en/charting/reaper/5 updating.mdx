As these tools improve, there will be updates to plugins and scripts. While you can update by downloading everything again, you can also just update the plugins and scripts.

## Updating everything
For any big update, downloading the entire pack is suggested. You update the same way you installed the REAPER charting files for the first time.
<Tabs>
  <TabItem label="Windows">
<a href="https://www.dropbox.com/scl/fi/d095opr4dforkodoyakkv/REAPER-Files-WIN.zip?rlkey=in0qshm2g4ng64w3angz8ehfq&st=vyiy4h7y&dl=1" target="_blank" rel="noopener noreferrer">[Click here to download the REAPER charting files `.zip` archive.]</a>.
    * a. Extract the `.zip` archive somewhere convenient.
    * b. Drag the `Common` folder into `DRAG 'Common' HERE` shortcut.
    * c. Drag `REAPER` into the `DRAG 'REAPER' HERE` shortcut.
    * d. Replace any files if you're asked.
    > <a href="https://www.dropbox.com/scl/fi/d095opr4dforkodoyakkv/REAPER-Files-WIN.zip?rlkey=in0qshm2g4ng64w3angz8ehfq&st=vyiy4h7y&dl=1" target="_blank" rel="noopener noreferrer"><Image src={explorerdrag} alt="Folders being dragged into shortcuts to other folders." /></a>
</TabItem>
<TabItem label="Linux">
Run the following command in a terminal:  
```bash
curl https://guides.milohax.org/downloads/chart/download-reaper-files.sh | bash
```
That's all!
> <Image src={archtermchartfiles} alt="A terminal downloading the REAPER charting files via curl." />
</TabItem>
</Tabs>
## Updating only plugins and scripts
If you're absolutely confident in your template and don't want to risk it being changed, this option only updates plugins and nothing else.  
The process is just like the initial download except the `.zip` file has a slightly different name.
<Tabs>
  <TabItem label="Windows">
<a href="https://www.dropbox.com/scl/fi/kofagfu0fp8ua5th9elmj/REAPER-Files-Update-WIN.zip?rlkey=450pm5976sxx094xvax0jcq4y&st=16jube72&dl=1" target="_blank" rel="noopener noreferrer">[Click here to download the REAPER Files Update `.zip` archive.]</a>.
    * a. Extract the `.zip` archive somewhere convenient.
    * b. Drag the `Common` folder into `DRAG 'Common' HERE` shortcut.
    * c. Drag `REAPER` into the `DRAG 'REAPER' HERE` shortcut.  
    * d. Replace any files if you're asked.
    > <a href="https://www.dropbox.com/scl/fi/kofagfu0fp8ua5th9elmj/REAPER-Files-Update-WIN.zip?rlkey=450pm5976sxx094xvax0jcq4y&st=16jube72&dl=1" target="_blank" rel="noopener noreferrer"><Image src={explorerdrag} alt="Folders being dragged into shortcuts to other folders." /></a>
</TabItem>
<TabItem label="Linux">
Run the following command in a terminal:  
```bash
curl https://guides.milohax.org/downloads/chart/update-reaper-files.sh | bash
```
That's all!
</TabItem>
