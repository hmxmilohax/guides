<Aside type="caution"><Image src={rpcs3nomap} alt="Una advertencia que dice que no mapeés este instrumento por el menu de 'Pads'." />  
¡NO mapeés este instrumento por el menú de `Mandos`!
</Aside>
<Aside type="caution">Menús de RPCN (para aceptar y mandar invitaciones para jugar en linea) causan que el juego se quede atorado.  
	Necesitas tener otra manera de controlar estos menús, como <a href="../../main#mandos" target="_blank" rel="noopener noreferrer">[un teclado de PC o un mando estándar.]</a></Aside>

* Las guitarras de Rock Band 1 **requieren su receptor propio para conectarse**.  
Las guitarras de Rock Band 1 tiene mástil de arce (color crema) con un receptor negro.  
![Receptor de guitarra de Rock Band para PlayStation 3 negro](/assets/instruments/adapters/ps3guitarrb1.png "Receptor de guitarra de Rock Band para PlayStation 3 negro")
Model#: `8221512`  
* Las guitarras de Rock Band 2 (y después) **requieren su receptor propio para conectarse.**  
Los receptores de Rock Band 2 y The Beatles: Rock Band son intercambiables y compatible con sus revisiones.  
![Receptor de guitarra de Rock Band para PlayStation 3 negro](/assets/instruments/adapters/ps3guitarrb2.png "Receptor de guitarra de Rock Band para PlayStation 3 negro")
Model#: `PGTSELEA2B`  
![Receptor de guitarra de Rock Band para PlayStation 3 gris](/assets/instruments/adapters/ps3guitarrb2gry.png "Receptor de guitarra de Rock Band para PlayStation 3 gris")
Model#: `PGTSELEA2B`  
![Receptor de guitarra de The Beatles: Rock Band para PlayStation 3](/assets/instruments/adapters/ps3guitarbeatles.png "Receptor de guitarra de The Beatles: Rock Band para PlayStation 3")
Model#: `PGTSELEA3B`

No necesitas configurar nada para usar este instrumento. Simplemente conecta el receptor a tu computadora ¡y es todo!
