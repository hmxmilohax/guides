<Aside type="caution">Menús de RPCN (para aceptar y mandar invitaciones para jugar en linea) causan que el juego se quede atorado.  
	Necesitas tener otra manera de controlar estos menús, como <a href="../../main#mandos" target="_blank" rel="noopener noreferrer">[un teclado de PC o un mando estándar.]</a></Aside>

* Tu batería MIDI necesita estar:
	* **en canal de MIDI 10.**
	* **estar configurada a el mapeo de GM.**  
Esto debe de ser lo que viene por defecto con mayoría de las baterías MIDI. Consulta el manual de tu batería para estar seguro.
* Necesitas mapear <a href="../../main#mandos" target="_blank" rel="noopener noreferrer">[mando dedicado para navegación]</a>. Puedes usar un teclado de PC o un mando estándar.
	* Puedes activar esto en `[Menú] > [Opciones] > [Configuración Deluxe] > [Controlador] > [Mando de Nav.]`
	><Image src={navpad} alt="El menú de mando de navegación dentro de los menús de Rock Band 3." />

## Instrucciones
**Si tu batería MIDI tiene un puerto de USB**, solo necesitas **conectarla a tu computadora**.  
><Image src={usbdrums} alt="El cerebro de la batería MIDI, mostrando un puerto de USB" />

**Si tu batería MIDI tiene un puerto de USB**, solo necesitas **conectarla a tu computadora**.  
><Image src={mididrums} alt="El cerebro de la batería MIDI, mostrando un puerto MIDI estilo DIN de cinco pines." />

Aquí esta un ejemplo de un adaptador de MIDI a USB. Mayoría tiene una luz LED integrada para mostrar activad.  
**Para verificar que esta todo enchufado correctamente, debes de ver <span title="Entrada de MIDI">"MIDI In"</span> parpadeando cuando le pegues a un parche**.  
><Image src={miditousb} alt="Una interfaz de MIDI a USB" />

**Si tienes una interfaz de audio, puede ser que ya tengas un puerto de MIDI** para tu computadora, ya que unas interfaces tienen puertos de MIDIs integrados.  
Por ejemplo, esta Scarlett tiene conexiones MIDI por detrás.  
><Image src={midifs} alt="El posterior de una Focusrite Scarlett, mostrando un puerto de USB y un salida/entrada MIDI por puerto DIN de cinco pines." />

Decide cual manera es mas conveniente para ti y conecta tu batería MIDI a tu computadora.

Después de eso, **haz click derecho en Rock Band 3** en RPCS3 **y luego en `Cambiar configuración personalizada`**.  
<Image src={rpcs3customconfigchange} alt="El menú de clic derecho de RPCS3, mostrando 'Change Custom Configuration' resaltado." />

Ahora, ve a la pestaña de `E/S`.  
<Image src={iod} alt="La configuración personalizada de I/O de Rock Band 3, mostrando 'Emulated MIDI Devices', tipo de dispositivo y selección de dispositivos resaltados en cuadros bronceados con contornos sólidos." />  
* ![Un cuadro bronceado con un contorno solido.](/assets/rb3pc/config/smalltan.png "Cuadro bronceado") : 
	* 🥁 **Cambia el `Dispositivo MIDI emulado` de `Teclado` a `Batería` y elige tu batería o interfaz MIDI en el menú desplegable junto a él**.

## Combos
Ya que baterías no tienen botones de mandos, necesitas usar ciertos combos para poder entrar a menús importantes.  
Aquí están los combos:
* `START`: Cierra el Hi-Hat, cierra el Hi-Hat, cierra el Hi-Hat, tarola
* `SELECT`: Cierra el Hi-Hat, cierra el Hi-Hat, cierra el Hi-Hat, aro de tarola
* Atajo para escoger canciones: cierra el Hi-Hat, cierra el Hi-Hat, cierra el Hi-Hat, bombo

Si quieres cambiar esto, mira la [[sección de configuración adicional]](#configuración-adicional)

## Configuración adicional
Puedes ajustar varias cosas en el archivo `rb3drums.yml`, que esta colocado en la carpeta llamada `config` dentro de donde tienes RPCS3.  
Abajo esta lo que hace cada opción. Necesitaras reiniciar el juego para que tus ajustes tomen efecto.

| **Ajuste** | **Descripción** |
|:----------:|:---------------:|
| `Pulse width ms: 30` | Duración (en MS) de cada nota en la batería. **Deja esto como esta.** |
| `Minimum velocity: 10` | La velocidad (intensidad) mínima para que una nota sea detectada.<br>Cualquier nota con velocidad menos de este valor no va ser contada.</br>|
| `Combo window in milliseconds: 2000` | Cuanto tiempo tienes para completar los combos de `START`, `SELECT` y el atajo para escocer canciones.<br>Valores están en milisegundos. Lee [[Combos]](#combos) para mas información.</br>|
| `Stagger cymbal hits: true` | Si le pegas a dos platillos al mismo tiempo, uno va ser demorado por el tiempo que tiene puesto en `Pulse width ms`.<br>**Deja esto como esta.**</br>|
| `Midi id to note override: ""` | Esto es para remapear lo que hace cada nota MIDI.<br>Lee [[Remapeando]](#remapeando) para mas información.</br>|
| `Combo Start: HihatPedal,HihatPedal,HihatPedal,Snare` | Este es el combo que emula el botón `START`.<br>Lee [[Combos]](#combos) para mas información.</br>|
| `Combo Select: HihatPedal,HihatPedal,HihatPedal,SnareRim` | Este es el combo que emula el botón `SELECT`.<br>Lee [[Combos]](#combos) para mas información.</br>|
| `Combo Toggle Hold Kick: HihatPedal,HihatPedal,HihatPedal,Kick` | Este es el combo que emula el atajo para escocer canciones.<br>Lee [[Combos]](#combos) para mas información.</br>|
| `Midi CC status: 176` | El byte que determina el estatus. Puede ser notado como `0xB0`.<br>**Deja esto como esta.**</br>|
| `Midi CC control number: 4` | El control continuo (CC) que esta a cargo del pedal de Hi-Hat.<br>**Deja esto como esta.**</br>|
| `Midi CC threshold: 64` | Esto es el punto medio del pedal de Hi-Hat, que separa si esta cerrado o abierto. |
| `Midi CC invert threshold: false` | Si lo cambias a `true`, invierte el punto de pedal mencionado arriba. |

### Remapeando
Si tu batería tiene partes que necesitan ser cambiadas, puedes usar `Midi id to note override: ""` para corregir notas.
* Para hacer esto:
	* Ve al sitio <a href="https://www.midimonitor.com/" target="_blank" rel="noopener noreferrer">[MIDI Monitor]</a>
	* Toca la parte que quieres re-configurar para encontrar su numero de nota MIDI (`Note #[numero]`).

El archivo usa estos nombres para notas:

<span title="Bombo">"<code>Kick</code>"</span>  
<span title="Pedal de Hi-Hat">"<code>HihatPedal</code>"</span>  
<span title="Tarola">"<code>Snare</code>"</span>  
<span title="Aro de tarola">"<code>SnareRim</code>"</span>  
<span title="Tom alto">"<code>HiTom</code>"</span>  
<span title="Tom bajo">"<code>LowTom</code>"</span>  
<span title="Tom de piso">"<code>FloorTom</code>"</span>  
<span title="Pedal de Hi-Hat abierto">"<code>HihatWithPedalUp</code>"</span>  
`Hihat`  
`Ride`  
`Crash`

Ejemplos de cambios comunes:
* El platillo azul esta configurado a `Note #51` y el platillo verde esta configurado a `Note #49` pero tu batería los tiene al revés.
	* `Midi id to note override: "49=Ride,51=Crash"` va a reversar los platillos para que estén sean mejor para Rock Band 3.
* Quieres usar tu Hi-Hat cerrada para amarillo y Hi-Hat abierta para azul.
	* `Midi id to note override: "46=Ride"`
