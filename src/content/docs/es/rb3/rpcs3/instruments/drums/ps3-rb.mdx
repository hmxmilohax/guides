<Aside type="caution"><Image src={rpcs3nomap} alt="Una advertencia que dice que no mapeés este instrumento por el menu de 'Pads'." />  
¡NO mapeés este instrumento por el menú de `Mandos`!
</Aside>
<Aside type="caution">Menús de RPCN (para aceptar y mandar invitaciones para jugar en linea) causan que el juego se quede atorado.  
	Necesitas tener otra manera de controlar estos menús, como <a href="../../main#mandos" target="_blank" rel="noopener noreferrer">[un teclado de PC o un mando estándar.]</a></Aside>

* Las baterías de Rock Band 1 pueden ser conectadas por USB. Las baterías de Rock Band 1 son negras con parches grises y no tienen conectores para platillos. 
* Las baterías de Rock Band 2 (y después) **requieren su receptor propio para conectarse.**  
Los receptores de Rock Band 2 y The Beatles: Rock Band son intercambiables y compatible con sus revisiones.  
![Receptor de batería de Rock Band para PlayStation 3 negro](/assets/instruments/adapters/ps3drumsrb2blk.png "Receptor de batería de Rock Band para PlayStation 3 negro")
![Receptor de batería de Rock Band para PlayStation 3 gris](/assets/instruments/adapters/ps3drumsrb2gry.png "Receptor de batería de Rock Band para PlayStation 3 gris")
![Receptor de batería de The Beatles: Rock Band para PlayStation 3](/assets/instruments/adapters/ps3drumsbeatles.png "Receptor de batería de The Beatles: Rock Band para PlayStation 3")

No necesitas configurar nada para usar este instrumento. Simplemente conecta el receptor a tu computadora ¡y es todo!
