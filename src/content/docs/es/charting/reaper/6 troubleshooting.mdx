<Tabs>
  <TabItem label="Windows">
* ¡REAPER me da `WASAPI: no se pudo activar dispositivo de entrada` cuando cambio a `WASAPI`!
  > <Image src={errordevicewin} alt="REAPER mostrando un error que dice 'There was an error opening the audio hardware. WASAPI Error: could not find input device'." />
  * `WASAPI: no se pudo activar dispositivo de entrada`: Es muy probable que tienes un micrófono conectado o el formato no coincide con la de tu salida (i.e. altavoces están a 48,000 Hz y el micrófono esta a 44,100 Hz).  
  Si es la primera, cambia `Canales de entrada` a `0` en la sección de `Dispositivos` en las Preferencias.  
  Si es la segunda, cambia el formato para que coinciden juntos.
* ¡No me carga el plugin para visualizar (RBN Preview)!
  > <Image src={errorpluginwin} alt="REAPER mostrando un error con RBN Preview." />
  * No deberías de seguir usando este plugin, pero si quieres ser terco, te faltan archivos de Microsoft Visual C++ Redistributable Runtimes.  
  <a href="https://kutt.it/vcpp" target="_blank" rel="noopener noreferrer">[Haz click aquí para descargarlos]</a>.  
</TabItem>
<TabItem label="Arch">
* ¡REAPER me da <span title="JACK: error creando cliente">"`JACK: error creating client`"</span> cuando uso `JACK`!
  > <Image src={errorjackarch} alt="REAPER mostrando un error que dice 'JACK: error creating client'." />
  #### Solución A:
  * Tienes `jack2` instalando y necesitas reemplazarlo con `pipewire-jack`.
    * Instala `pipewire-jack` ejecutando este comando en la terminal.
        ```bash
        sudo pacman -S pipewire-jack
        ```
          > <Image src={archpwjack} alt="Una terminal descargando pipewire-jack de pacman." />
  #### Solución B:
  * Cambia a otro dispositivo como `PulseAudio` o `ALSA`.
    > <Image src={settingsaudioarch} alt="Las preferencias de REAPER, mostrando la sección de Dispositivos." />
</TabItem>
<TabItem label="Fedora">
* Tengo un error de `Error de ejecución del script` cuando trato usar CAT o VenueGen que dice <span title="ModuloNoEncontradoError: No hay un modulo que se llama 'tkinter'">"`ModuleNotFoundError: No module named 'tkinter'`"</span>
  > <Image src={errorreascriptfed} alt="REAPER mostrando un error con ReaScript." />
  * Instala "python3-tkinter"
    ```bash
    sudo dnf install python3-tkinter
      ```
        > <Image src={fedtermtk} alt="Una terminal descargando python3-tkinter de DNF." />
* ¡REAPER me da `Hubo un error al abrir el hardware de audio` cuando uso `JACK`!
  > <Image src={errordevicegnu} alt="REAPER showing an error that says 'Three was an error opening the audio audio hardware'." />
  * Cambia a otro dispositivo como `PulseAudio` o `ALSA`.
    > <Image src={settingsaudioarch} alt="Las preferencias de REAPER, mostrando la sección de Dispositivos." />
</TabItem>
