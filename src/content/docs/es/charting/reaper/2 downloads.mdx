## Instalando REAPER
<Tabs>
  <TabItem label="Windows">
<Steps>
1. <a href="https://www.reaper.fm/download.php" target="_blank" rel="noopener noreferrer">[Descarga la versión actual de 64-bits de REAPER.]</a>.
    * **¡Por favor NO actives la opción de `Portable install` en el instalador!**  
    > <a href="https://www.reaper.fm/download.php" target="_blank" rel="noopener noreferrer"><Image src={webreaperwin} alt="La pagina para descargar REAPER, mostrando el botón de descarga para Windows. El cursor esta sobre el boton verde de descarga." /></a>

2. <a href="https://www.sws-extension.org" target="_blank" rel="noopener noreferrer">[Descarga la versión actual de 64-bits de SWS / S&M Extension.]</a>.
    * **¡Por favor deja la opción de `Python ReaScript support` activada cuando lo instales!**  
    > <a href="https://www.sws-extension.org" target="_blank" rel="noopener noreferrer"><Image src={webswswin} alt="La pagina para descargar la extension SWS/S&M Extension para Windows. El cursor esta sobre el boton de 'Download x64' en la sección de Windows." /></a>

3. <a href="https://www.python.org/downloads" target="_blank" rel="noopener noreferrer">[Descarga la versión actual de Python 3 si no la tienes.]</a>.
    * **¡No descargues `Python install manager`!**
    * **Python 2 no es lo mismo y no va a funcionar**.
    > <a href="https://www.python.org/downloads" target="_blank" rel="noopener noreferrer"><Image src={webpython} alt="La pagina para descargar Python con el cursor sobre el botón de 'get the standalone installer for Python 3.14.2'." /></a>

4. Cuando termine, abre el instalador.  
En el instalador:  
    * Activa <span title="Agregar python.exe a PATH">"`Add python.exe to PATH`"</span>
    * Haz click en <span title="Instalar ahora">"<code>Install now</code>"</span>  
    > <Image src={pyinstall} alt="El instalador de Python. 'Add python.exe to PATH' y 'Install now' estan resaltados en cuadros bronceados con contornos sólidos." />

5. <a href="https://www.dropbox.com/scl/fi/d095opr4dforkodoyakkv/REAPER-Files-WIN.zip?rlkey=in0qshm2g4ng64w3angz8ehfq&st=vyiy4h7y&dl=1" target="_blank" rel="noopener noreferrer">[Haz click aquí para descargar los archivos para REAPER de una carpeta `.zip`.]</a>.
    * a. Extrae el archivo `.zip` a un lugar conveniente.
    * b. Arrastra la carpeta de `Common` al acceso directo que se llama de `DRAG 'COMMON' HERE`.
    * c. Arrastra la carpeta de `REAPER` al acceso directo que se llama `DRAG 'REAPER' HERE`.
    * d. Reemplaza todos los archivos si te pregunta.
    > <a href="https://www.dropbox.com/scl/fi/d095opr4dforkodoyakkv/REAPER-Files-WIN.zip?rlkey=in0qshm2g4ng64w3angz8ehfq&st=vyiy4h7y&dl=1" target="_blank" rel="noopener noreferrer"><Image src={explorerdrag} alt="Las carpetas siendo arrastrados a carpetas en formato de acceso directo." /></a>
</Steps>
</TabItem>
<TabItem label="Linux">
<Tabs>
<TabItem label="Arch">
<Aside type="caution">¡Esta parte esta diseñada para instalaciones de Arch que son de x86-64/AMD64 64-bits **y no para ARM**!</Aside>
Instalando REAPER en Arch es demasiado fácil. ¡Se hace con tres comandos! Solo usas estos comandos:
<Steps>
1. Instala REAPER con SWS y ReaPack. TK y Unzip se descargar y se instalaran si faltan o actualizaran si ya los tienes.
    ```bash
    sudo pacman -S reaper sws reapack tk unzip
    ```
    > <Image src={archreaperdl} alt="Una terminal preguntado descargando REAPER y otras cosas. Le pregunta al usuario quiere continuar." />
2. Necesitas crear un <span title="Enlace simbólico">"Symlink"</span> para que SWS funcione con REAPER.
    ```bash
    ln -s /usr/lib/sws/reaper_sws-x86_64.so ~/.config/REAPER/UserPlugins/
    ln -s /usr/lib/sws/sws_python64.py ~/.config/REAPER/Scripts/
    ```
    > <Image src={archsymlink} alt="Una terminal creando un enlace simbólico de 'usr/lib' a '~/.config'." />
3. Después de que termine de instalar, necesitas descargar e instalar los archivos para chartear con REAPER.
    Ejecuta este comando en la terminal:  
    ```bash
    curl https://guides.milohax.org/downloads/chart/download-reaper-files.sh | bash
    ```
    Esto va a descargar y colocar todos los archivos que necesitas para chartear con REAPER.
    > <Image src={archtermchartfiles} alt="Una terminal descargando los archivos para chartear con REAPER con el comando de curl." />
</Steps>
</TabItem>
<TabItem label="Fedora">
<Aside type="caution">¡Esta parte esta diseñada para instalaciones de Linux de x86-64/AMD64 64-bits **y no para ARM**!</Aside>
<Steps>
1. <a href="https://www.reaper.fm/download.php" target="_blank" rel="noopener noreferrer">[Descarga la versión actual de 64-bits de REAPER compatible con tu Distro]</a>.
    > <a href="https://www.reaper.fm/download.php" target="_blank" rel="noopener noreferrer"><Image src={webreapernix} alt="La pagina para descargar REAPER, mostrando el botón de descarga para Windows." /></a>

2. <a href="https://www.sws-extension.org" target="_blank" rel="noopener noreferrer">[Descarga la versión actual de 64-bits de SWS / S&M Extension]</a>.
    > <a href="https://www.sws-extension.org" target="_blank" rel="noopener noreferrer"><Image src={webswsnix} alt="La pagina para descargar la extension SWS/S&M Extension para Linux. El cursor esta sobre el boton de 'Download x64' en la sección de Linux." /></a>

3. <a href="https://reapack.com" target="_blank" rel="noopener noreferrer">[Descarga la versión actual de 64-bits de ReaPack]</a>.
    > <a href="https://reapack.com" target="_blank" rel="noopener noreferrer"><Image src={webreapackgnu} alt="La pagina para descargar la extension ReaPack. El cursor esta sobre el boton de 'x86 64-bit' en la sección de Linux." /></a>

4. Ve al archivo de REAPER que se descargo para extraerlo y luego entra a la carpeta que sale después de extraerlo.
    > <Image src={fedreaperdl} alt="Una carpeta extraída de REAPER con el cursor sobre el icono." />

5. Ejecuta la script de de `install-reaper.sh`.
    > <Image src={fedreaperins} alt="El cursor esta sobre la script de 'install-reaper.sh'." />

6. En la primera parte de la script, elige `I` para instalar REAPER.
    > <Image src={fedtermreaper1} alt="Una terminal preguntado si quieres abrir o instalar REAPER, leer el README o agregar integración con el escritorio." />

7. En la siguiente parte, te preguntara si quieres instalar REAPER a la carpeta del sistema o del usuario.  
    **¡Debes de instalar REAPER a la carpeta del usuario con `2`!**  

    Después de eso, te preguntara si quieres activar integración de archivos.  
    Esto se recomienda para fácilmente poder abrir archivos `.RPP` (archivos de proyecto de REAPER).  
    Elige `2` para activar esto.

    Finalmente, te preguntara si quieres continuar.  
    Elige `Y` y deja que termine instalando REAPER.
    > <Image src={fedtermreaper2} alt="Una continuación de la instalación de REAPER. Esta preguntando de lo que se hablo arriba." />

8. Abre REAPER después de que termine de instalar.
    > <Image src={fedreaperopen} alt="REAPER en la búsqueda de programas. El cursor esta sobre el icono del programa." />

    Si REAPER te pregunta si quieres elegir una dispositivo de audio ("audio device driver"), elige `No` por ahora.
    > <Image src={settingsaudioalert} alt="REAPER alertando el usuario que no ha elegido un dispositivo de audio." />

    Si REAPER te alerta que no se puede abrir los dispositivos de audio ("audio hardware"), ignóralo por ahora.
    > <Image src={errordevicegnu} alt="REAPER alertando el usuario que no se puede abrir el dispositivo de audio." />

9. Ve a <span title="Opciones">"`Options`"</span> > <span title="Mostrar la ruta del recurso REAPER en el explorador/buscador">"`Show REAPER resource path in explorer/finder...`"</span>
    > <Image src={fedreaperpath} alt="REAPER in the search results, ready to open. The cursor is over the program icon." />

10. Después de que se abra la carpeta con los recursos de REAPER, cierra REAPER.
    > <Image src={fedreapercls} alt="El cursor sobre el botón de cerrar REAPER." />

11. Deja la carpeta de recursos de REAPER abierta y abre otro navegador de archivos. Ve a donde se descargo el archivo de SWS y extráelo.  
Ahora, ve a la carpeta que salio.
    > <Image src={fedswsdl} alt="Una carpeta extraída de SWS con el cursor sobre el icono." />

12. Arrastra la carpetas dentro de la carpeta de SWS a la carpeta de recursos de REAPER, escribiendo sobre cualquier cosa.
    > <Image src={fedswsdnd} alt="Las carpetas de SWS siendo arrastrados a la carpeta de recursos de REAPER." />

13. Deja la carpeta de recursos de REAPER abierta y con el otro navegador, busca el archivo que se descargo de ReaPack.  
    Arrástralo ese archivo a la carpeta de `UserPlugins` que esta en la carpeta de recursos de REAPER.
    > <Image src={fedreapackdnd} alt="Arrastrando el archivo de ReaPack a 'UserPlugins' dentro de la carpeta de recursos de REAPER." />

14. Después de todo eso, necesitas descargar e instalar los archivos para chartear con REAPER.
    Ejecuta este comando en la terminal:  
    ```bash
    curl https://guides.milohax.org/downloads/chart/download-reaper-files.sh | bash
    ```
    Esto va a descargar y colocar todos los archivos que necesitas para chartear con REAPER.
    > <Image src={fedtermchartfiles} alt="Una terminal descargando los archivos para chartear en REAPER." />
</Steps>
</TabItem>
